Vienna ELECTRO

Find the best parties in Vienna!

Overview

Vienna ELECTRO is a mobile app which allows the user to find the best parties in Vienna. The user can filter parties by current day, week, or weekend.

Process

  1. Identify User Persona
  2. Build the Sitemap
  3. Create User Flows
  4. Ideation
  5. Design and mockups - in Figma
  6. Iterate pervious steps until design is complete and tested
